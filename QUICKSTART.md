# Quick Start

```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/marknorgren/dotfiles-public/main/install)"
source ~/.zshrc
```

Useful commands after setup:

```bash
just verify
dh
g status
week
now
uuid
```

Private or work setup belongs in an ignored `local/` file or a separate private
overlay repo.
